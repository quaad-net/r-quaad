Plan-o-gram

I was tasked with creating an application that would produce a simple plan-o-gram (a visual layout) of items optimally positioned on shelves given various product and shelf dimensions. The product dimensions were stored in a data table while shelf levels and dimensions where taken in the form of user input(this would vary as desired by clients). The plan-o-gram could then be used as a guide for the initial setup of products at the client's location.

The application asks users to point to a file containing a clients desired products (which typically ranges from hundreds to thousands), then also takes the client's shelf dimensions. Given those sets of data, it outputs the plan-o-gram.



Outline

  • Application takes user input for dimensions and products file.
  • Sets products on shelves optimally given product and shelf dimensions.
  • Outputs data for plan-o-gram.
  • Places and styles finished plan-o-gram in an Excel workbook.

Product Usage

This is a module that was part of a larger application written to analyze usage of various products. The functions in the module are used to get the past 12 months of usage of a specified product (moving usage). The resulting data can be viewed and/or exported as a file.
